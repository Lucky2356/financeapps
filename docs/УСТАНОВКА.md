# Установка «Финансового помощника» (Windows)

Приложение работает на **Windows 10 и Windows 11** (64-bit). Все ваши данные
хранятся только на вашем компьютере — без облака и без передачи третьим лицам.

## 1. Скачать установщик

Откройте страницу **Releases** репозитория и скачайте файл вида:

```
Финансовый помощник_1.0.0_x64-setup.exe
```

## 2. Запустить установку

1. Дважды кликните по скачанному файлу.
2. Windows может показать синее окно **«SmartScreen защитил ваш компьютер»** —
   это нормально для приложений без платной подписи издателя. Нажмите
   **«Подробнее» → «Выполнить в любом случае»**.
3. Следуйте шагам установщика. Приложение устанавливается для текущего
   пользователя и не требует прав администратора.
4. При первом запуске, если в системе нет компонента **WebView2**, он будет
   автоматически загружен и установлен (нужен интернет один раз).

## 3. Первый запуск

При первом открытии появится короткое обучение. Можно:

- **«Загрузить пример»** — заполнить приложение демонстрационными данными,
  чтобы осмотреться;
- **«Начать с нуля»** — и пройти по шагам «Быстрого старта» на главной
  (добавить счёт → записать операцию → задать бюджет → создать цель).

Демо-данные можно удалить в любой момент: **Настройки → Очистить все данные**.

## 4. Обновление

Скачайте новый установщик со страницы Releases и запустите его — он обновит
приложение, ваши данные сохранятся.

## 5. Резервная копия

На странице **Импорт** можно выгрузить полную резервную копию (JSON) и затем
восстановить её на другом компьютере или после переустановки.

## Системные требования

- Windows 10 версии 1809+ или Windows 11, 64-bit
- ~150 МБ свободного места
- Интернет — только для первичной установки WebView2 и для онлайн-котировок
  акций (раздел «Инвестиции»); всё остальное работает офлайн

## Важно

Раздел «Инвестиции» использует данные Московской Биржи (MOEX) и носит
справочный характер. Это не индивидуальная инвестиционная рекомендация.
